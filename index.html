How we used BlackRcok, Cisco Spark, SparkPost, Microsoft Cognitive Services, Prediction Markets, Google Cloud Storage

We used BlackRock's API to track an investment group's stock's three month growth. Using SparkPost's API, an email will be sent to the group with the stock's information and said improvement. The email will also contain a link to a group video chat powered by Cisco Spark. This video chat, stored on Google Cloud Storage, will collect data using Microsoft Cognitive Services facial recognition to detect the group's emotion during the call. Using the data from the video chat and the stock's three month growth, the Celebrate Stock Return analysis will give tips on how to proceed with that stock. We then implemented OpenTable's API to allow the group to celebrate accordingly based on the collected data.
